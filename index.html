Cloud & DevOps Project

Project Overview

This project demonstrates a fully automated cloud deployment using AWS and DevOps practices. It includes a containerized application deployed on AWS ECS Fargate with an Application Load Balancer and CloudFront distribution for global access.

Technologies & Services Used

AWS ECS Fargate
Amazon ECR
AWS CloudFront
Application Load Balancer
GitHub Actions
Docker
AWS CDK

Step by Step Flow

Code is pushed to GitHub → GitHub Actions workflow triggers → Docker image is built and pushed to ECR → ECS Fargate runs the container → ALB routes traffic → CloudFront distributes globally → Users can access the app quickly and securely.

Benefits

  • Fully automated deployment reduces manual effort.
  • Global fast access through CloudFront.
  • Scalable and reliable cloud infrastructure.
  • Demonstrates professional cloud & DevOps skills.
  • Secure and repeatable infrastructure using code.
